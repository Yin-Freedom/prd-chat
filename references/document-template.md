## 编写指南

### 优先级定义

| 优先级 | 说明 |
|--------|------|
| 高 | 核心功能，必须实现 |
| 中 | 重要功能，建议实现 |
| 低 | 增强功能，可延后实现 |

### 功能类型分类

| 类型 | 说明 |
|------|------|
| 新增 | 创建新数据 |
| 查询 | 查看数据列表或详情 |
| 修改 | 编辑已有数据 |
| 删除 | 删除或归档数据 |
| 导入 | 批量导入数据 |
| 导出 | 导出数据 |
| 审批 | 审批流程操作 |
| 统计 | 数据统计分析 |
| 其他 | 其他类型操作 |

### 角色类型分类

| 类型 | 说明 |
|------|------|
| 系统角色 | 系统管理相关角色，如管理员 |
| 业务角色 | 业务操作相关角色，如业务员 |
| 外部角色 | 系统外部用户，如客户 |

## 文档版本规则

- **主版本号**：重大变更，如产品方向调整
- **次版本号**：功能增删，如新增模块
- **修订号**：细节修改，如字段调整

示例：V1.0 → V1.1 → V1.2 → V2.0
